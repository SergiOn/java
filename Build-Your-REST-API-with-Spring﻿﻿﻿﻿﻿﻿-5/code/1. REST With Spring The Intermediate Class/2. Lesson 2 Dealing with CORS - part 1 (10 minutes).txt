1. Goal
Learn about and understand the Same Origin Policy restriction of the browser, CORS and how to relax that restriction in Spring. 

2. Lesson Notes
The Same Origin Policy
The Same Origin Policy is an important restriction implemented by all major browsers in order to not allow JS code from making requests against a different origin.

An Origin:

URI scheme
Hostname
Port Number
